# Archetype
---
The shows we love most are the ones with characters we resonate with. Archetype helps you find new film and TV titles across different genres based on personality traits — even your own!

**Live app**: https://archetype-8mt.pages.dev/

## Features
---
**Cross-genre character mapping:** Ask what a character you love would look like in a completely different genre. Archetype finds characters from other titles who share the same drives, flaws, and character arcs.
- "What if Walter White was dropped into a high fantasy epic?"

**Match your personality:** Select your Myers-Briggs type or describe your own personality traits, pick a genre, and get recommendations for characters who think and feel like you do.
- ENFJ + Comedy
- "Introverted, stubborn, dry humor, loyal once committed" + Thriller

**Personality analysis:** Ask to go dive deep into any character and get a focused breakdown of their psychology: core motivation, behavior under pressure, and what makes them dramatically compelling.
- "Tell me more about that {ABC}"

## How it works
---
Each film and TV character in the catalog has a personality blurb generated by a Llama 3.3 model on Workers AI. These blurbs are embedded and stored in a vector index. When you send a message, a Cloudflare Workflow:

1. Extracts your intent and builds a search query
2. Runs semantic search over the character catalog
3. Filters out any characters you named (they're the anchor, not the recommendation)
4. Writes a personalized rationale for each match explaining the personality parallel

## Stack
---
- **Frontend:** React + Vite, deployed on Cloudflare Pages
- **Backend:** Cloudflare Worker with Workflows and Durable Objects
- **AI:** Llama 3.3 70B via Cloudflare Workers AI
- **Vector search:** Cloudflare Vectorize + BGE-M3 embeddings
- **Session memory:** Durable Objects (SQLite-backed, 40-message history)
- **Real-time updates:** WebSocket push via Durable Object — no polling

Film genre and poster images from TMDB.

## Running locally
---
**Prerequisites:** Node 18+, a Cloudflare account, and a TMDB API key.

```bash
npm install
npx wrangler login
```

Create the Vectorize index (first time only):
```bash
npx wrangler vectorize create archetype-characters --dimensions=1024 --metric=cosine
```

Build the character catalog (requires Cloudflare credentials for LLM blurb generation):
```bash
TMDB_API_KEY=xxx CLOUDFLARE_API_TOKEN=xxx CLOUDFLARE_ACCOUNT_ID=xxx node scripts/build-catalog.mjs
```

Start the worker (`.dev.vars` is already configured with a local seed secret):
```bash
npm run dev
```

Seed Vectorize:
```bash
curl -X POST http://localhost:8787/api/seed \
  -H "Authorization: Bearer archetype-local-dev-seed"
```

Start the frontend:
```bash
npm run dev:web
```

Open [http://localhost:5173](http://localhost:5173).
